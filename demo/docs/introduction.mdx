# Introduction

Welcome to the **HMS Stack** documentation. This comprehensive guide will help you understand, set up, and work with the Hospital Management System stack.

## What is HMS Stack?

HMS Stack is a modern, full-stack solution for hospital management systems, built with:

- **Next.js** for frontend applications
- **Spring Boot** for backend services
- **Agent Framework** for intelligent code analysis and fixes
- **Modern tooling** for documentation and development
